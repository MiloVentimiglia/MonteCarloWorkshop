Propagating samples through the model function¶

Assume the measurement model, i.e. the relation between the input quantities and the measurand, to be given as $$ Y = f(X_1,X_2,\ldots,X_N) $$

with joint PDF associated with the input quantities given as $g_{\mathbf{X}}(\mathbf{\xi})$.

The aim of the Monte Carlo method is the propagation of samples from the PDF $g_{\mathbf{X}}(\mathbf{\xi})$ through the measurement model $f()$ as a means to calculate samples from the PDF $g_{Y}(\eta)$.

Basically, two types of implementation have to be considered: for-loops and vectorisation.

Monte Carlo using for-loops¶

This is the most easiest way of implementing Monte Carlo for a given function $f()$.

Y = np.zeros(number_of_runs)
    X1,X2,...,XN = draw_from_gX(number_of_runs)
    for k in range(number_of_runs):
        Y[k] = model(X1[k],X2[k],...,XN[k])

Monte Carlo using vectorisation¶

For scripting languages, such as Python, Matlab or LabView, a for-loop is often very inefficient. Instead, a vectorisation of the model function should be considered.

For instance, let the model function be $$ Y = X_1 \sin (2\pi X_2) - X_3^2 $$

The vectorized Monte Carlo method for this model is given as

X1,X2,X3 = draw_from_gX(number_of_runs)
    Y = X1*np.sin(2*np.pi*X2) - X3**2

# solution for exercise 3.1
from scipy.stats import uniform, t

# random draws from input quantities
draws = 10000
X1 = uniform.rvs(loc=-0.8, scale=1.6, size=draws)
X2 = t.rvs(11, loc=3.4, scale=0.5, size=draws)

# evaluate model function
Y = X2**2 / (sqrt(1-X1**2))

figure(figsize=(16,6))
subplot(131)
hist(X1,bins=100, edgecolor="none")
subplot(132)
hist(X2,bins=100, edgecolor="none")
subplot(133)
hist(Y,bins=100, edgecolor="none")
suptitle("PDFs associated with input and output quantities",fontsize=20);

# solution for exercise 3.2
from scipy.optimize import curve_fit

# model function
def modelfun(t,a,b,c):
    return a * exp(-b*t) * sin(c*t)

# GUM-like uncertainty evaluation
noise = sigma*ones_like(x)
phat,uphat = curve_fit(modelfun, t, x, sigma=noise)

# GUM Monte-Carlo uncertainty evaluation
draws = 1000

pMC = zeros((draws, 3))  # set up matrix to store MC results
success= []   # list of successfull fits
failed = []   # list of failed fits

for k in range(draws):
    xMC = x + random.randn(len(x))*sigma  # draw from input quantities
    # use try-except structure to catch failed fits
    try:
        pMC[k,:] = curve_fit(modelfun, t, xMC, sigma=noise)[0]
        success.append(k)
    except RuntimeError:
        failed.append(k)        

# use fancy indexing to select only results where the fit was successful
phatMC = mean(pMC[success],axis=0)
uphatMC= cov(pMC[success],rowvar=0) 

figure()
errorbar(range(1,4), phat, sqrt(diag(uphat)), fmt="o", label="GUM result")
errorbar(range(1,4), phatMC, sqrt(diag(uphatMC)), fmt="s", label="Monte Carlo result")
legend()
xlim(0,4)
xticks([1,2,3], ["a", "b", "c"]);
tick_params(which="both",labelsize=20)

# calculate Monte-Carlo model function outputs
t_long = linspace(t[0], t[-1], 1000)
Y = zeros((len(t_long), draws))
for k in range(len(success)):
    pars = pMC[success[k]]
    Y[:,k] = modelfun(t_long, *pars)

figure()
errorbar(t, x, sigma*ones_like(x),fmt="o", label="measured")
plot(t, modelfun(t, *phat), "gd", label="modelled")
plot(t_long, Y, "g", alpha=0.05)
legend();
